rate(stars, user, dimension = nil)

Submits a new rate. Accepts a hash of tipical Ajax request.

Example: 

  # Articles Controller
  def rate
    @article = Article.find(params[:id])
    @article.rate(params[:stars], current_user, params[:dimension])
    # some page update here ...
  end
